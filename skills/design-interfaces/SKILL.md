# Design Interfaces Best Practices

Esta skill proporciona guías completas para el diseño de interfaces de usuario modernas, accesibles y responsivas.

## Categorías Principales

### 1. Principios de UX
- Jerarquía visual
- Consistencia
- Feedback del sistema
- Affordance y signifiers

### 2. Componentes UI
- Botones, inputs, modales
- Navegación
- Formularios
- Cards y containers

### 3. Diseño Responsivo
- Mobile-first
- Breakpoints
- Fluid typography
- Grid systems

### 4. Accesibilidad
- WCAG guidelines
- ARIA labels
- Focus management
- Color contrast

### 5. Sistemas de Diseño
- Design tokens
- Component libraries
- Theming
- Dark mode
